# Excel-HR-Analytics-Dashboard
HR analytics dashboard built in Excel using pivot tables, formulas and KPI tracking to analyze workforce trends and HR metrics.


Project Overview

This project analyzes HR data using Excel to track workforce metrics such as headcount, department distribution, and compensation trends. The dashboard provides quick insights for HR decision making.

Business Problem

HR teams require structured reporting to monitor workforce metrics but often rely on manual reports. This project demonstrates how Excel dashboards can automate reporting and improve visibility.

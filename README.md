## 📁 Projects

### 📘 Project 1 — Depreciation Calculator

**Files:** `Project-1(Depreciation-Calculator-Excel).xlsx` · `Project-1(Depreciation-Calculator).pptx`

#### Business Problem
Businesses need to track asset depreciation accurately for financial reporting and tax compliance. Manual calculations across multiple assets are error-prone and time-consuming.

#### What Was Built
An automated Excel depreciation calculator that:
- Computes depreciation using multiple methods (Straight Line, Reducing Balance)
- Tracks asset value over useful life
- Generates depreciation schedules for reporting
- Supports multiple assets simultaneously

#### Excel Features Used
`IF formulas` `Array formulas` `Data Validation` `Conditional Formatting` `Charts` `Named Ranges`

#### Key Output
Structured depreciation schedules that can be directly used for financial reporting and asset management.

### 📘 Project 2 — Financial Analysis

**Files:** `Project-2 (Financial_Analysis_Report).xlsx` · `Project-2(Financial Analysis Project Documentation).pdf`

#### Business Problem
Investment decisions require rigorous financial modelling — evaluating loan scenarios, comparing investment returns, and understanding NPV/IRR outcomes across different assumptions.

#### Analysis Performed
- NPV (Net Present Value) calculation across investment scenarios
- IRR and XIRR for irregular cash flow analysis
- EMI calculation for loan scenario comparison
- PV (Present Value) analysis for annuities
- Interest rate sensitivity modelling
- Investment return summaries for decision support

#### Excel Functions Used
`NPV` `IRR` `XIRR` `PV` `PMT` `FV` `RATE` `NPER` `IF` `Data Tables`

#### Key Insights
- IRR comparisons reveal which investment scenarios outperform the cost of capital
- EMI sensitivity tables show how rate changes impact total repayment cost
- NPV analysis supports go/no-go investment decisions with clear breakeven thresholds

### 📘 Project 3 — Bare International Performance Analysis

**Files:** `Project-3(Bare International Analysis).xlsx` · `Project-3(Bare International Analysis).pptx`

#### Business Problem
A global organisation needed to evaluate performance scores across regions and standardise evaluation to identify high performers and areas needing improvement.

#### Analysis Performed
- Performance score analysis across regions and evaluators
- Regional comparison using pivot tables
- High-performing region identification
- Performance grading framework development
- Standardised evaluation template creation

#### Excel Features Used
`Pivot Tables` `VLOOKUP` `INDEX MATCH` `Conditional Formatting` `Charts` `Grading formulas`

#### Key Insights
- Performance scores vary significantly across regions, with clear top and bottom performers
- Standardised grading framework eliminates evaluator bias in scoring
- Regional benchmarking enables targeted improvement planning
